# 📝 مشروع إدارة المهام (Tasks)

نظام بسيط لإدارة المهام اليومية (To-Do List) تم تطويره باستخدام PHP (OOP) و MySQL، يساعد المستخدم على تنظيم مهامه بسهولة ومرونة.

---

## 🚀 نظرة عامة

مشروع **Tasks** هو تطبيق ويب بسيط يهدف إلى إدارة المهام اليومية، حيث يمكن للمستخدم إضافة المهام، تعديلها، حذفها، وعرضها بشكل منظم.

---

## ✨ المميزات

* ➕ إضافة مهمة جديدة
* ✏️ تعديل المهام
* ❌ حذف المهام
* 📋 عرض جميع المهام
* 🔐 (اختياري) نظام تسجيل دخول للمستخدمين

## 🛠️ التقنيات المستخدمة

* PHP (البرمجة الكائنية OOP)

## ⚙️ طريقة التشغيل

### 1. تحميل المشروع

```id="x7p1kd"
git clone https://github.com/Dev-s3d/Tasks.git
```

### 2. نقل المشروع إلى السيرفر المحلي

مثال:

* XAMPP → داخل مجلد `htdocs`

### 3. إنشاء قاعدة البيانات

* افتح phpMyAdmin
* أنشئ قاعدة بيانات جديدة باسم: `tasks_db`

### 4. استيراد قاعدة البيانات (إن وجدت)

* قم باستيراد ملف `.sql` داخل قاعدة البيانات

### 5. تعديل إعدادات الاتصال

افتح الملف:

```id="m2n8qp"
config/config.php
```

وعدّل البيانات:

```php id="k8v3sj"
$host = "localhost";
$dbname = "tasks_db";
$username = "root";
$password = "";
```
